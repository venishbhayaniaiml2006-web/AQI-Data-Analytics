🌍 Air Quality Index (AQI) Data Analysis Project

📌 Project Overview

This project focuses on analyzing city-wise Air Quality Index (AQI) data using Python. The goal is to understand pollution patterns, identify trends, and generate meaningful insights through data cleaning, exploratory data analysis (EDA), and visualization techniques.

---

🎯 Objectives

- Analyze AQI levels across different cities
- Identify pollution trends over time
- Perform data cleaning and preprocessing
- Apply EDA techniques to extract insights
- Visualize air quality patterns using graphs and charts

---

🛠️ Technologies & Tools Used

- Programming Language: Python
- Libraries:
  - Pandas (Data Manipulation)
  - NumPy (Numerical Operations)
  - Matplotlib (Data Visualization)
  - Seaborn (Statistical Visualization)
- Environment: Jupyter Notebook / VS Code

🔍 Workflow / Methodology

1. Data Collection

- Collected AQI dataset containing city-wise pollution data

2. Data Cleaning

- Removed missing values
- Handled inconsistent data
- Converted data types where required

3. Exploratory Data Analysis (EDA)

- Statistical summary of AQI values
- Identified highest and lowest polluted cities
- Detected trends and patterns

4. Data Visualization

- Bar charts for city comparison
- Line graphs for AQI trends over time
- Heatmaps for correlation analysis

---

📊 Key Insights

- Certain cities show consistently high AQI levels (high pollution)
- Seasonal variations impact air quality
- Industrial and urban areas tend to have higher AQI
